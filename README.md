
Blockchain To-Do List (JavaScript)

A simple blockchain-based To-Do List application where every task is stored as a block in a blockchain.
This project is beginner-friendly and helps understand core blockchain concepts using JavaScript.

Features

Add tasks to a blockchain (each task = a block)

Each block contains:

Index

Task description

Hash

Previous block hash

Verifies data integrity using hashing

Detects tampering using isValid() method
